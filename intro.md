# ICH Modelling Power Markets

Welcome to the **ICH Modelling Power Markets** course. This course teaches you how to model and analyse integrated power markets using [PyPSA](https://pypsa.org) (Python for Power System Analysis).

## What You Will Learn

By the end of this course, you will be able to:

- Set up and solve power system optimisation models in Python
- Build multi-node AC network models from Excel input files
- Run capacity expansion planning with carbon constraints
- Simulate electricity market dispatch, unit commitment, and price formation
- Model quadratic cost functions and bidding strategies
- Implement rolling-horizon day-ahead and intra-day market simulations
- Add reserve requirements to market models
- Apply these techniques to real regional case studies (South Africa and Zambia)
