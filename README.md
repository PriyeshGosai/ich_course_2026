# ICH Modelling Power Markets

This repository contains course materials for the **Integrated Power Markets** course developed for the [International Centre for Hydropower (ICH)](https://www.hydropower.org/ich).

## Course Website

The full course is available at: **https://priyeshgosai.github.io/ich_course_2026/**

For an overview of the course content, lessons, and structure, see the [course introduction](intro.md).

---

## About the Course

This course was developed for ICH's Integrated Power Markets programme and teaches power system modelling using [PyPSA](https://pypsa.org) (Python for Power System Analysis). It covers electricity market theory and hands-on computational modelling, progressing from foundational concepts through to regional case studies in southern Africa.

## Quick Start

All notebooks are designed to run in **Google Colab** — no local installation required. Visit the [course website](https://priyeshgosai.github.io/ich_course_2026/) and click the rocket icon on any lesson page to launch it directly in Colab.

## Disclaimer

The models used in this course are developed exclusively for educational purposes and have been intentionally simplified. They should not be used for policy analysis, operational assessments, or any substantive study of real power systems.
